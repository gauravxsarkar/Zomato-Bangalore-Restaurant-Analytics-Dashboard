# 🍽️ Zomato Bangalore Restaurant Analytics Dashboard

*A data-driven dashboard to explore restaurant trends, ratings, and customer preferences in Bangalore.*

---

## 📌 About the Project

This project is an interactive data analytics dashboard built using Zomato’s Bangalore restaurant dataset. It helps uncover insights about restaurant types, customer ratings, pricing patterns, and location-based trends.

The goal is simple: turn raw data into meaningful insights that can help understand the food ecosystem in a city like Bangalore — whether you're a data enthusiast, aspiring analyst, or just someone curious about food trends.

---

## 🎯 Purpose

The purpose of this project is to:

* Analyze restaurant distribution across Bangalore
* Understand what factors influence ratings (like online ordering, table booking, etc.)
* Explore pricing trends and restaurant categories
* Build a clean, intuitive dashboard for quick decision-making

---

## 🛠️ Tech Stack

This project combines multiple tools commonly used in real-world data roles:

* **Python** → Data cleaning & exploratory data analysis (EDA)
* **Pandas, NumPy** → Data manipulation
* **Matplotlib / Seaborn** → Visualization during EDA
* **SQL (PostgreSQL)** → Data querying & transformation
* **Power BI** → Dashboard creation & interactive visualization

---

## 📊 Data Source

The dataset used in this project is publicly available on Kaggle:
👉 https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants

It contains detailed information about restaurants in Bangalore including:

* Ratings
* Votes
* Cost for two
* Location
* Online ordering & table booking availability
* Restaurant type/category

---

## ✨ Features & Highlights

### 📈 Key Metrics Overview

* Total number of restaurants
* Average ratings and votes
* Average cost for two
* Percentage of highly rated restaurants

### 🔍 Feature Impact Analysis

* Compare ratings based on:

  * Online ordering availability
  * Table booking availability
* Helps identify what actually improves customer experience

### 🍕 Restaurant Type Insights

* Distribution of restaurant categories (Quick Bites, Casual Dining, etc.)
* Clear visual breakdown using charts like donut and treemap

### 📍 Location-Based Analysis

* Top areas in Bangalore by restaurant count
* Helps identify food hubs and dense restaurant clusters

### 🎛️ Interactive Filters

* Filter by restaurant type
* Filter by cost category (Low / Medium / High)
* Makes exploration dynamic and user-driven
